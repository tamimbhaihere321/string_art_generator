# Professional String Art Generator

Generate workshop-ready handmade string art:

- Accurate nail coordinates in millimeters
- Printable drill template with mm grid
- Optimized wrapping instructions
- Live thread mapping preview
- High-resolution final render

---

## LOCAL RUN

pip install -r requirements.txt
streamlit run app.py

---

## DEPLOY TO STREAMLIT CLOUD

1. Push project to GitHub
2. Open Streamlit Cloud
3. Click New App
4. Select repo + branch
5. Choose app.py
6. Deploy
7.
